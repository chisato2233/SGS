新增反应机制

### 属性类型
暂定这些
庶风 - 风
离火 - 火
冥寒 - 冰
息壤 - 土
虩霆 - 雷
太阴 - 阴
太阳 - 阳

未来有可能加入更多属性

角色的判定阶段会后发先致地执行反应栈中的印记结算：

风 / 火 ：每回合消散一层，上限五层
冰 / 雷：不会消散，上限三层
土：不会消散，上限一层

火：消散时，对附着者造成一点火焰伤害
雷：层数达到三层时，移除印记，对附着者造成三点雷电伤害
冰：判定阶段，若本印记达到三层，移除印记，跳过本回合出牌阶段和弃牌阶段


### 单反应公式
1风 + 1火/1冰/1雷 = 扩散：转移受反应者印记层数到【周围】
1土 + 1火/1冰/1雷 = 结晶：增加一点护甲

1火 + 1冰 = 融化：无效果
1火 + 任意数量的雷 = 1雷 + 1火 = 超载：立刻对附着者造成一点雷电伤害

任意数量的雷 + 任意数量的冰 = 超导：将附着者的护甲削减至原先的50%

任意数量的火 + 1阳 = 1阳 + 任意数量的火 = 废锁：废除附着者所有技能，直到其下个回合结束后

任意数量的冰 + 1阴 = 1阴 + 任意数量的冰 = 隐匿：【移除游戏】直到本轮结束

1阴 + 1阳 = 中和，移除所有非阴阳印记，然后模四张牌

### 反应规则
涉及阴阳参与的优先级高于一般反应，阴阳中和的优先级最高，同优先级角色的印记栈后发先致地与触发元素反应

伤害量不与参与反应的元素量挂钩，没有特殊提及，属性伤害的元素量均为1点

### 术语
1. 属性伤害：造成伤害并触发反应，即使反应没有消耗玩触发者的元素量也不会留下印记
2. 属性附着伤害：造成伤害并触发反应，反应后残留的元素量加入印记栈
3. 属性附着：触发反应，反应后的残留的元素量会加入印记栈
4. 连锁反应：有些反应触发同样会造成属性伤害，有些反应会给其他反应栈加入标记，他们都会与剩余印记继续做反应，这些单反应仍然算作一次反应结算的结果
